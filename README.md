# GO Tac Toe

this is a kata i repeat in new languages to play a game of tic-tac-toe either Player vs. Player, or Player vs. Machine (Minimax algorithm)

## Getting started
`cd /src`
`go build`
`go test`

## Current Features
- Play a game PvP
 -- inputs are "0,0" .. "2,2"
